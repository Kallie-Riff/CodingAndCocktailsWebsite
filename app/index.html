The Code

You've seen the paint & wine classes, now it's time to get techy and learn web development basics over trendy cocktails.
As an intro to programming, you'll get an overview of the history of building for the web, and jump right into the state of things today - building single page applications using tools like github, yeoman, sass, and gulp. We like to think of it as the essential web developer tool kit.
When you've got the basics down, you'll learn how to create and deploy your own blog using resources like Jekyll and Twitter's Bootstrap to show off your epic coding prowess. And once you've hit the Ballmer Peak courtsey of crafted cocktails, it's to infinity and beyond in writing code and learning to mentor others.

The Cocktails

We couldn't be more excited to partner with bartender Laura Wagner of Local Pig Westport. She brings the freshest local ingredients together with liquors from local vendors to create a custom cocktail menu to get your code on to.

The Mentors

As a woman, learning to program can be intimidating! We've recruited fantastic female mentors to create a welcoming learning environment and answer those questions you may be too scared to ask!
These ladies write code for their day job and can't wait to help you get started. They're kind of like super heroes, with great taste!
